## Default Variables

### nfs_exports

List of available exports

#### Default value

```YAML
nfs_exports: []
```

#### Example usage

```YAML
nfs_exports:
  - path: /data/bar
    host: 192.168.0.0/24
    options:
      - rw
      - sync
      - no_subtree_check
  - path: /data/baz
    network: 192.168.0.0/24
```
